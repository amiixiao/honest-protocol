# Contributing to Honest Protocol

> "Honesty over Fluency. Determinism over Probability."

---

## [ENGLISH]

### 1. Our Philosophy
This project is dedicated to building a deterministic future for AGI. We value structural integrity over generative flair. Any contribution must align with the core principle: **AI must have the courage to remain silent when uncertain.**

### 2. How to Contribute
- **Logic Refinement:** We welcome proposals that enhance the mathematical definition of the confidence score $P(f)$.
- **Implementation:** Extensions for specific LLM frameworks are encouraged, provided they respect the Dynamic Honesty Ladder (DHL).
- **Audit Process:** All Pull Requests (PRs) undergo a rigorous logic audit. We prioritize reliability over speed.

## [中文版]

### 1. 我们的哲学
本项目致力于为 AGI 构建一个确定性的未来。我们认为结构完整性高于生成式的华丽。任何贡献都必须符合核心原则：**AI 必须有勇气在不确定时保持沉默。**

### 2. 如何贡献
- **逻辑完善：** 我们欢迎任何旨在增强置信度评分 $P(f)$ 数学定义的提案。
- **框架实现：** 鼓励为特定 LLM 框架开发扩展，前提是必须遵循“动态诚实阶梯 (DHL)”。
- **审计流程：** 所有拉取请求 (PR) 都将经过严格的逻辑审计。我们认为可靠性远比开发速度更重要。
