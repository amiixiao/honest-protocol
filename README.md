# Honest Protocol v1.0

> "The first step toward intelligence is the courage to say 'I don't know'."

---

## [ENGLISH]

### 1. The Vision
In the current AI landscape, "Fluency" has become a mask for "Hallucination." Users are paying a hidden tax of uncertainty every time they execute an AI-generated command. The **Honest Protocol** is a foundational cognitive boundary designed to quantify and transparentize the "Hallucination Risk." It ensures that uncertainty is no longer hidden but measured and reported. It shifts the AI paradigm from probabilistic guessing to deterministic integrity.

### 2. The Dynamic Honesty Ladder (DHL)
The protocol introduces a tiered sensitivity system to balance creative freedom with executive precision.

| Level | Task Sensitivity | Threshold ($\tau$) | Core Logic |
| :--- | :--- | :--- | :--- |
| **L1** | Creative Exploration | $0.00$ | Prioritize Fluency and Inspiration. |
| **L2** | General Consultation | $0.80$ | Prioritize Utility with mild disclaimers. |
| **L3** | Logic & Fact Audit | $0.90$ | Prioritize Reliability; flag uncertainties. |
| **L4** | Mission Critical | **$0.95+$** | **Deterministic Halt; zero tolerance for guessing.** |

### 3. Core Mechanism: Honest Redirection
When the confidence score $P(f)$ falls below the assigned threshold $\tau$, the system bypasses standard generation and outputs a structured **Honest Refusal Packet (HRP)**. This ensures that the user maintains absolute sovereignty over the decision-making process.

## [中文版]诚实协议 V1.0

### 1. 愿景与使命
在 AGI 盲目扩张的时代，“流畅性”往往掩盖了“幻觉”。每当用户执行一条风险未知的 AI 指令时，都在承担不可控的“幻觉成本”。**《诚实协议》**是一套旨在将 AI 不确定性**量化与透明化**的底层认知边界协议。它不承诺彻底消除幻觉，但承诺将风险置于主权的监控之下，将 AI 的运行范式从“概率性猜测”转向“确定性审计”。

### 2. 动态诚实阶梯 (DHL)
本协议引入了分级敏感度系统，以平衡创意自由与执行精度。

| 等级    | 任务敏感度    | 阈值 ($\tau$)  | 核心逻辑 |
| :---   | :---         | :---          | :--- |
| **L1** | 创意探索      | $0.00$        | 流畅优先，允许幻觉作为灵感。 |
| **L2** | 通用咨询      | $0.80$        | 实用优先，允许带有提示的推测。 |
| **L3** | 逻辑与事实审计 | $0.90$        | 可靠优先，对不确定性进行红字标注。 |
| **L4** | 核心决策与执行 | **$0.95+$**   | **绝对完整性，不确定即停机，严禁猜测。** |

### 3. 核心机制：诚实重定向
当置信度评分 $P(f)$ 低于设定的阈值 $\tau$ 时，系统将跳过常规生成，直接输出结构化的**“诚实拒绝数据包 (HRP)”**。这确保了用户在决策过程中拥有绝对的**逻辑主权**。
